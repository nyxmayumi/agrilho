# 🌱 Agro Forte - Futuro Sustentável

Um site moderno e responsivo dedicado à sustentabilidade agrícola, promovendo o equilíbrio entre produção e preservação do meio ambiente.

## 📋 Características

✨ **Design Moderno e Responsivo**
- Layout totalmente responsivo (mobile, tablet, desktop)
- Navegação intuitiva com menu hamburger
- Animações suaves e transições elegantes

🎨 **Temas Visuais**
- Paleta de cores verde (tema ecológico)
- Gradientes profissionais
- Tipografia clara e legível

⚡ **Interatividade**
- Formulário de contato funcional
- Animação de contadores de estatísticas
- Scroll suave entre seções
- Observador de intersecção para animações de entrada

📱 **Mobile-First Approach**
- Otimizado para todos os tamanhos de tela
- Menu responsivo com hamburger
- Toque otimizado para dispositivos móveis

## 🎯 Seções do Site

1. **Home (Hero)** - Apresentação principal com CTA
2. **Sobre** - Informações sobre a Agro Forte
3. **Soluções** - 6 soluções principais de sustentabilidade
4. **Benefícios** - 4 benefícios principais
5. **Estatísticas** - Números de impacto
6. **Contato** - Formulário de contato e informações

## 🚀 Como Usar

### Opção 1: Abrir Localmente
1. Clone ou baixe os arquivos
2. Coloque os 3 arquivos (index.html, styles.css, script.js) na mesma pasta
3. Abra `index.html` no navegador

### Opção 2: Usar em um Servidor
```bash
# Usando Python 3
python -m http.server 8000

# Usando Python 2
python -m SimpleHTTPServer 8000

# Usando Node.js (http-server)
npx http-server
